我的工具列表

袁旭佳(Aaron)
2026-04-03
tools

以下是个人Mac 电脑上常用的工具，做一个分类收集：

开发工具

  • warp - AI 命令行工具，平替 iTerm2
  • Alfred 5 - macOS 效率启动器，支持快捷搜索和工作流自动化
  • Bartender 5 - macOS 菜单栏图标管理工具，隐藏不常用的图标让菜单栏更清爽
  • Tower - 可视化 Git 分支管理工具
  • uTools - 各类混合工具，例如 JSON 格式化
  • Snipaste - 截图工具

终端相关

  • brew - macos平台包管理工具
  • oh-my-zsh - 终端配置工具，替换默认的 bash 配置
  • tig - 终端下的 Git 日志可视化查看工具
  • autojump - 终端目录快速跳转工具，通过记忆常用路径实现智能跳转
  • tree - 目录树结构查看工具

网络代理

  • Charles - 网络抓包与代理调试工具，用于 HTTP/HTTPS 请求分析
  • ProxyMan - macOS 平台强大的网络抓包与 API 调试工具
  • KeepingYouAwake - 防止系统休眠

写作笔记

  • 幕布 - 极简大纲笔记工具，支持一键生成思维导图
  • Logseq - 本地基于日期的 Markdown 笔记软件
  • Notion - All-in-one 笔记与协作工具，支持数据库、看板等多种内容组织形式
  • MacDown - macOS 原生的 Markdown 编辑器，实时预览

投屏工具

  • scrcpy - Android 设备投屏到 Mac
  • 鸿蒙设备投屏 - 使用 hdc 命令行工具实现鸿蒙设备屏幕投射到 Mac

其他

  • DiffMerge - 跨平台的文件对比与目录对比合并工具

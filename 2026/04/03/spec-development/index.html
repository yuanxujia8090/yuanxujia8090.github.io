Spec 应用和实践

Spec Coding 是一种规范驱动开发（Spec-Driven Development，SDD）的 AI Coding 开发范式，核心思路是将模糊需求的 prompt 转换为具体的标准规范文档（如技术方案 design.md、任务清单 tasks.md）来引导、驱动 AI 进行开发，本质上是上下文构建。在编写代码之前，先让开发者和AI对齐需求，从而解决传统”氛围编程”中的架构漂移、上下文丢失、技术债务和返工频繁等问题

Claude Code 使用攻略

常用命令

命令 说明
/help 显示帮助信息
/clear 清除对话历史
/fast 切换快速模式（更快输出）
/plan 进入计划模式
/commit 创建 git commit
/review-pr 审查 Pull Request

初始化命令

命令 说明
claude init 初始化 Claude Code 项目
claude auth 认证 Claude Code
claude config 配置 Claude Code
